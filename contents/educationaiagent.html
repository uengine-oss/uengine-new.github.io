AI 에이전트 플랫폼 개발 교육과정

오픈소스 기반 엔터프라이즈 AI 에이전트 시스템 구축 마스터 코스
교육 신청하기

과정 개요

본 교육과정은 프로세스 GPT라는 오픈소스 AI 에이전트 플랫폼을 활용하여 기업용 AI 에이전트 시스템을 구축하고 운영할 수 있는 전문 개발자를 양성하는 종합 교육 프로그램입니다. BPMN 기반의 비즈니스 프로세스 엔진과 최신 AI 기술을 결합하여 실무에서 바로 적용 가능한 실전 중심의 커리큘럼을 제공합니다.

교육 목표
• 오픈소스 AI 에이전트 플랫폼을 활용한 기업용 솔루션 개발 역량 확보
• BPMN 기반 비즈니스 프로세스 엔진과 AI 에이전트의 통합 구현
• Vector RAG 및 Graph RAG 기반 지능형 검색 시스템 구축
• Mem0 기반 장기메모리 시스템 설계 및 구현으로 개인화된 AI 에이전트 개발
• LangGraph를 활용한 복잡한 에이전트 워크플로우 설계 및 구현
• A2A(Agent-to-Agent) 및 MCP(Model Context Protocol) 프로토콜을 통한 에이전트 간 통신 구현
• 온프레미스 환경에서의 AI 시스템 구축 및 운영 실무 경험

이런 분들을 위한 과정입니다

👨‍💻

초급 이상의 파이썬 개발자

🤖

AI/ML 기술을 활용한 비즈니스 솔루션 개발에 관심이 있는 개발자

👥

기업용 AI 에이전트 시스템 구축을 담당하는 개발팀

🏢

오픈소스 기반 AI 플랫폼 도입을 검토하는 기업의 기술진

주요 커리큘럼

Chapter 1. Agentic AI 기초 & ProcessGPT

주요 학습 내용 및 활동(개념)

• 기존 AI vs Agentic AI
• 마이크로소프트 에이전틱 AI 도입 3단계

실습

• ProcessGPT 환경 구축
• 에이전트 선언/학습/프로세스 생성
🤖 결과물: 작동하는 업무 자동화 에이전트 1개

Chapter 2. MCP 기반 툴 구현

주요 학습 내용 및 활동(개념)

• MCP 아키텍처
• FastMCP 프레임워크

실습

• Python Code Interpreter MCP 툴 개발
• 클라이언트 디버깅
🤖 결과물: ProcessGPT 연동된 Python 실행 환경 MCP 툴

주요 학습 내용 및 활동(개념)

• Agent-to-Agent 통신
• mem0 메모리 시스템

실습

• ProcessGPT 프로세스 A2A 변환
🤖 결과물: 학습 능력을 가진 Python 코딩 A2A 에이전트

Chapter 4. 멀티 에이전트 시스템 구축

주요 학습 내용 및 활동(개념)

• CrewAI 프레임워크

실습

• Human-in-the-Loop

주요 학습 내용 및 활동(통합)

• ProcessGPT ↔ MCP ↔ A2A ↔ CrewAI 전체 연동

실습

• Docker Compose 전체 스택 실행
• Kubernetes 배포

교육 정보

본 교육과정은 AI 기반 비즈니스 자동화의 새로운 패러다임을 제시하며, 참가자들이 차세대 AI 에이전트 시스템의 전문가로 성장할 수 있도록 체계적인 로드맵을 제공합니다.

교육 신청하기
교육기간
1 Day 교육
9:30 ~ 17:30 (총 7시간)
교육방식
이론 + 실습 + 프로젝트
수강료
300,000원 (VAT별도)
수료증
프로세스 GPT 공인 개발자 자격증 발급

서비스 제공 기간 : 서비스는 결제 완료 후, 1개월 이내 제공됩니다. 결제는 교육 시작일 기준 2주 전부터 결제 가능합니다.
